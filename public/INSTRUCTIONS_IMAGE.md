# Instructions pour ajouter l'image du header

## Image requise

**Nom du fichier** : `Copie de Copie de Copie de Copie de Sans titre.png`

## Emplacement

Placez l'image dans le dossier `public/` à la racine du projet :

```
public/Copie de Copie de Copie de Copie de Sans titre.png
```

## Après avoir ajouté l'image

1. Redémarrez le serveur de développement si nécessaire
2. Videz le cache du navigateur (Ctrl+Shift+R ou Cmd+Shift+R)
3. L'image devrait apparaître dans le header hero

## Note

Si l'image ne s'affiche pas, vérifiez :
- Que le nom du fichier correspond exactement (attention aux espaces et majuscules)
- Que l'image est bien dans le dossier `public/`
- Que le serveur de développement a été redémarré
