# Image Hero pour Jessica CONTENTIN

## Instructions

Pour ajouter l'image hero du site vitrine de Jessica CONTENTIN :

1. Placez l'image dans le dossier `public/`
2. Nommez l'image : `Copie de Copie de Copie de Copie de Sans titre.png`
3. Format recommandé : JPG ou PNG
4. Dimensions recommandées : 1920x1080px ou plus (format paysage)
5. L'image sera automatiquement utilisée dans le header hero section

## Emplacement du fichier

```
public/Copie de Copie de Copie de Copie de Sans titre.png
```

## Description de l'image attendue

L'image doit représenter :
- Une femme rayonnante qui sourit
- Ambiance chaleureuse avec lumière dorée
- Tissu translucide et bracelets (selon l'image fournie)
- Couleurs chaudes (or, ambre, beige)

L'image sera affichée en plein écran dans le header avec le texte "Ensemble, révélons votre talent" et le CTA "Prendre rendez-vous" superposés.
